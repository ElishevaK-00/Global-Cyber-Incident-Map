# 🌍 Interactive Global Cybersecurity Incident Map & Dashboard

> 🚀 **Live Interactive Application:** [Click here to view the live interactive map dashboard!](https://ElishevaK-00.github.io/Global-Cyber-Incident-Map/)

## Project Overview
This repository contains a full-stack programmatic geospatial intelligence dashboard that transforms raw international cybersecurity research into a live, interactive web application. The platform maps 10 high-impact geopolitical cyber events, infrastructure intrusions, and defense policy milestones across explicit global coordinate points.

## 🛠️ The Technical Stack & Data Engine
- **Front-End Deployment:** Leverages **GitHub Pages** to host a standalone, lightweight, responsive HTML application (`index.html`).
- **Geospatial Engine:** Utilizes the **Leaflet.js** framework via open-source Content Delivery Networks (CDNs) to load responsive global base maps and coordinate tracking.
- **Dynamic Mapping Logic:** Loops through multi-dimensional research vectors, applying programmatic color-coded categorization markers based on legal tiers and incident profiles.

## 🚦 Interactive Marker Protocol
- 🔴 **Red Pins (International Violations):** Major unprovoked structural breaches violating international cyber norms (e.g., *Storm-0558 U.S. State Dept Cloud Intrusion*, *APT28 Outlook Zero-Day Exploit*).
- 🟡 **Orange Pins (Questionable Conflicts):** Complex cyber warfare and regional gray-zone espionage operations (e.g., *Operation FlightNight in India*, *South Korean Covert Agent Identity Leak*).
- 🟢 **Green Pins (Defensive Policy/Reform):** Landmark legislative overhauls and military restructuring designed to fortify cyber infrastructure (e.g., *Australia's Cyber Security Bill 2024*, *Germany's CIR Military Branch Elevation*).
