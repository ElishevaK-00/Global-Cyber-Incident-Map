# Interactive Global Cybersecurity Incident Map

## Project Overview
This repository contains a programmatic geospatial visualization engine transforming structured cybersecurity research into an interactive cloud map. The map catalogs 10 major global cyber events, infrastructure intrusions, and defense policy overhauls occurring across different global coordinates.

## Methodology & Analytics Stack
- **Geospatial Processing:** Translates country data points into latitude and longitude spatial vectors.
- **Data Ingestion:** Maps attributes for each coordinate marker, including Attack Vector, Attacker Profiling, System Impact, and Policy Tier Classification.
- **Layer Visualization:** Utilizes interactive Python modules (`Folium`) to render interactive map overlays directly verifiable on the web ecosystem.

## Tracked Research Hotspots
- **North America:** 2023 Microsoft Exchange Cloud Intrusion (Espionage tracking).
- **Western Europe:** Germany's 2024 Cyber Military Branch elevation; Poland/Czech Republic critical Outlook vulnerability vectors.
- **Middle East & Asia-Pacific:** UAE credential theft tracking (APT34); India's multiple energy sector vulnerabilities; Australia's historic national cybersecurity legislation framework.

## Core Technical Skills Displayed
- Spatial Data Analysis & Geocoding Mapping
- Structured Data Modeling (Translating text reports to parameter vectors)
- Scripting & Metadata Management (Python, Open-Source Mapping Modules)
